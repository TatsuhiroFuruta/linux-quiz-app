# Linuxコマンド練習 ~grep/sed/awkをマスターしよう~

## 概要
Linux のターミナル操作で用いられる grep, sed, awk コマンドを練習するアプリです。これらのコマンドは、ログファイルやファイル、コマンドの出力の情報を抽出したり加工したりする際に用いられます。初級者、中級者、上級者の3コース用意されており、実際に入力して出力結果を答え合わせすることで、実践的に学ぶことができます。時間を計測できるタイムアタックモードも設けられています。

## 作りたい理由
大学院時代、Pythonで機械学習の計算コードを書いていたというのもあり、Linuxコマンドを触れていましたが、 grep, sed, awkの3種類のコマンドをうまく使いこなせていなかったのが唯一の心残りでした。今現在、Webエンジニアを目指すためにプログラミング学習に励んでいます。Webアプリケーションが正常に動作していない原因を特定する際に、アプリ内部に関するログやサーバーのログ、ネットワーク状況など調査する機会が将来的に訪れる可能性が考えられます。以上の背景から、これらのコマンドを身につけるのは重要ではないかと思い、サービスを立ち上げることにしました。
